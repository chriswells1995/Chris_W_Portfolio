BrewBuzz

Search for breweries! Write and read reviews! By utilizing Passport, users can create an account with an encrypted password. When signed in, users can write reviews for their favorite breweries, and keep track of all the reviews they've written. But even without signing in, users can search for breweries by name or location, and see what has been said already. We also have each brewery's website, their google map directions, their phone number, and we allow users to share their experience to facebook or twitter with the click of a button! Our SQL database holds over 7000 breweries, and it's growing every day!

SquigglePig

A party game for 3 or more Players! Draw images, give them a funny caption, and vote which player's was the funniest! By utilizing Socket io, players can create a game room by setting a code and sharing it with friends. Multiple game rooms can be played simoltaneously without interfering with eachother. By Utilizing Canvas, players can draw using either a mouse or touch screen, and send their images to the other players' devices. Built in REACT.
